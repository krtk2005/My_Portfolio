# karthick-portfolio

This is my personal portfolio website showcasing my skills, projects, and experience as a frontend developer. The site is fully responsive, designed with modern UI principles, and includes:

✔ **Sections:** Home, About, Skills, Projects, Contact  
✔ **Technologies Used:** HTML, CSS, JavaScript, ScrollReveal animations  
✔ **Features:** 
   - Responsive design for all devices
   - Animated scroll effects
   - Downloadable resume
   - Contact form for easy communication

Future updates will include backend integration using **PHP & MySQL** for the contact form.
